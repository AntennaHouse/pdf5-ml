﻿1 Overview

This document describes how to integrate PDF5-ML plug-in with oXygen XML Editor.

﻿2 Downloading PDF5-ML plug-in from GitHub

Download or clone PDF5-ML plug-in from GitHub repository into your local file system.

[Fig.1]  Downloading or cloning PDF5-ML plug-in

﻿3 Confirm your oXygen XML Editor DITA version setting

Open your oXygen XML Editor DITA setting by clicking [Options]-[Preferences] and select [DITA]

[Fig.2]  Confirming which DITA-OT is selected.

Confirm your oXygen XML Editor DITA-OT folder. (Abbreviated as "[DITA-OT]")

DITA Open Toolkit Toolkit location
Bult-in DITA-OT 1.8.5 [oXygen Install Folder]/frameworks/dita/DITA-OT
Bult-in DITA-OT 2.x (with support for DITA 1.3 and Lightweight DITA) [oXygen Install Folder]/frameworks/dita/DITA-OT2.x

﻿4 Copy plug-in folder into [DITA-OT]/plugins folder

Copy "com.antennahouse.pdf5.xml" folder from the download into the [DITA-OT]/plugins folder.

☞ NOTE: This operation needs administrator authority.

﻿5 Plug-in integration

  1. Invoke oXygen XML Editor with administrator authority.
    If you use Windows select oXygen XML Editor icon in the desktop and from context menu select "Run as administrator".
  2. Open any ditamap file.
  3. Click [Document]-[Transformation]-[Configure Transformation Scenario(s)...]
  4. Check [Run DITA-OT Integrator].
  5. Click [Apply associated(1)] button.

﻿6 Modify transformation scenario for PDF5-ML plug-in

  1. From oXygen XML Editor Configure Transformation Scenario, duplicate the "DITA Map PDF".

    [Fig.3]  Duplicate the "DITA Map PDF" Schenario.

  2. Modify the duplicated scenario

    • Modify name from "DITA Map PDF - Copy" to "DITA Map PDF - PDF5-ML".
    • Select "Project options" or "Global options" according to your decision. If the former is selected, it can be used only this project. If the latter is selected it can be used from any project.
    • Modify "transtype" parameter to "pdf5.ml".
    • Add "ahf.dir" parameter and set your Antenna House Formatter install folder.
    • [Optional] Add "ahf.opt" parameter and set your favorite Antenna House Formatter option setting file path.
    [Fig.4]  Edit transformation scenario.

    ☞ NOTE: For more details about Antenna House Formatter setting file, refer to Option Setting File Help.

﻿7 Test the transformation scenario

  1. Invoke oXygen XML Editor and open any ditamap file which you want to generate PDF file.
  2. Click [Document]-[Transformation]-[Configure Transformation Scenario(s)...]
  3. Select "DITA Map PDF - PDF5-ML" scenario.
  4. Click [Apply associated]
  5. The ant will be invoked and if there is no error, the generated PDF file will be opened after build process.
[Fig.5]  Run transformation scenario.
